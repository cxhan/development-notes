# Development-notes


## 总览

这是一个 vue vuepress vuepress-creator 搭建的vue静态内容管理网站。

### 包含模块

- 杂项
- 前端
- 算法
- Java
- 系统运维

### 安装依赖

```bash
npm install
```

### 启动本地开发环境

```bash
npm run dev
```

### 构建生产环境

```bash
npm run build
```
